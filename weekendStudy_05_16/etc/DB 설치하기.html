[1] DB 설치하기

  • 컴퓨터 이름, 사용자 이름, 파일 경로 등에 한글이 포함되면 안 됨
  • port 번호 : 컴퓨터 내에서 구동되는 서비스 식별용 번호

1. Oracle XE와 SQL Developer 설치하기

1.1. Oracle XE

  • Oracle
    • DBMS(DataBase Management System), DB 관리 시스템 프로그램
    • 엄밀히 말하면 DB와 DBMS는 다르지만, 보통 따로 구분하지 않고 사용함
  • Oracle home : 오라클이 설치된 폴더 주소(oraclexe/app/oracle/product/11.2.0/server)
  • Oralce XE : Express Edition, 무료 버전, 교육용, 개인용(오라클 사용법을 익히는 용도)
  • Oracle XE 설치 후 SQL command line에서 연습용 계정 설정
    • SQL> conn system/zxcv1234;
      시스템 계정으로 접속, 비밀번호 zxcv1234는 설치 시 지정한 비밀번호
    • SQL> alter user hr account unlock;
      hr 계정 잠금해제
    • SQL> alter user hr identified by zxcv1234;
      hr계정의 비밀번호를 zxcv1234로 설정
    • SQL> conn hr/zxcv1234;
      hr 계정으로 DBMS 접속

1.2. SQL Developer

  • SQL Developer
    • SQL문 작성을 도와주는 에디터 프로그램
    • 이클립스와 마찬가지로 압축만 풀면 실행 가능
    • 우리는 이클립스가 있는 폴더에 같이 넣어주자
